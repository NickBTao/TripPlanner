Trip Planner

Two web Forms allow user to Register and Plan a Vacation with Various options and prices
Pure FrontEnd (HTML, CSS, Javascript)

Created by: Nicolas Beattie
Last Updated: august 5th, 2022

---------------------------------------------------------------------------------

DESCRIPTION
A basic exercise on front end design work
A first page allows users to Register, the second, plan a trip, the third is a success message.

NOTA BENE:
- Contains ONLY Front end implementation. No persistence (if reloaded, user has to start over from scratch)

FEATURES
- Basic Validation on Email and Passwords during registration using javascript.
- Trip Planner Form calculates and auto-refreshes cost estimate based on user inputs.

SETUP
Download folder, double click index.html
